# ghostty-shaders

Custom GLSL shaders for [Ghostty](https://ghostty.org) terminal.

## Shaders

- **lava-lamp-bg.glsl** — Slowly drifting color washes. Lava lamp vibes.
- **watercolor-bg.glsl** — Layered brushstrokes on textured paper.

## Usage

Add to your Ghostty config (`~/.config/ghostty/config`):

```
custom-shader = /path/to/shader.glsl
```

Pairs well with `background-opacity` for transparency.
